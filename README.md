# mini Chat app via Google Sheets API <br>
## How to send messages
First of all you only need `AppsScript.js` if you want to create it in your own sheet.
On the directory you downloaded or pasted `receiver.ps1` and `sender.ps1` right click and open in terminal
<br>
<img width="203" height="34" alt="image" src="https://github.com/user-attachments/assets/ed2e36b6-948e-4b2a-bb88-5ae3e09e16af" /> <br>
This will open a PowerShell terminal in that same directory. <br>
Now press `alt` + `shift` + `+` to split the terminal in two vertically or `alt` + `shift` + `-` to do so horizontally. <br>
in the first one execute
```
.\sender.ps1
```
or `.\` + the name and extension of the file if renamed <br>
on the other one
```
.\receiver.ps1
```
or `.\` + the name and extension of the file if renamed
<hr><br>
Now you just have to follow to file instructions.
